# Bean & Brew Sales Analytics Pipeline

An end-to-end Business Intelligence project demonstrating the complete analytics workflow from synthetic data generation to executive dashboard reporting.


## Project Overview

This project simulates first-quarter (Q1 2026) sales for a fictional coffee shop chain, **Bean & Brew Coffee Co.**

The objective was to build a complete Business Intelligence pipeline similar to those used in industry.

The project covers:

- Data Generation using Python
- Data Cleaning using Pandas
- Database Loading with SQLite
- Business Analysis using SQL
- Dashboard Development using Power BI
- Version Control using Git & GitHub

---


## Business Questions Answered

The dashboard answers questions such as:

- What is the total revenue?
- Which stores generated the highest sales?
- Which products contributed the most revenue?
- Which product categories perform best?
- What are the busiest sales hours?
- How does revenue change over time?
- What is the average transaction value?
- How many units were sold?

---

## Key Insights

Example findings include:

- Q1 revenue exceeded **$60K**.
- Coffee products generated the highest revenue.
- Morning hours experienced the highest transaction volume.
- Ottawa achieved the highest store revenue.
- More than 13,000 products were sold during the quarter.
